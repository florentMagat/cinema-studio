Pour réalisation cette application web, j'ai utilisé :

   - React 18.2 (+ react-dom & react-router-dom)
   - Vite
   - Typescript
   - Tailwindcss

Pour la tester la procédure est la suivante :

   - `npm i` ou `yarn install`
   - `npm run dev` ou `yarn dev`
   - `http://localhost:5173/`
